Encapsulation in regards to coding is the idea of grouping data and code that operates on that data together into one unit. The only thing that should be exposed to other programmers is a well-defined interface to control misuse of the code by other programmers. This is the basic idea behind function and classifications such as public and private. The goal should be to make the code as easy to use as possible with minimal chance of someone messing things up on accident.
