Abstraction in regards to code is the concept of purposefully hiding data and its implementation details. When working on even slightly larger than basic projects, it quickly becomes a problem to keep all the specific workings of code in your head as you develop, and so it becomes very useful to abstract the previously made code into general higher level ideas. This is one of the basic concepts behind functions which allow for more complex operations to be hidden away and controlled by just calling the function name which is much easier to understand and work with at a glance.
