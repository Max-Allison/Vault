Object Orientation is a framework for modeling applications and software based around the idea of sets of objects that interact with each other. It seeks to create very flexible and modular code that can be changed or added upon easily with minimal rewriting of already existing code. Object Oriented Programming is primarily made up of [[Objects]] which fuse data (i.e. variables) and functions (a.k.a methods) into one entity. These objects replace global-level functions as the primary method of [[Encapsulation]] and [[Abstraction]].
